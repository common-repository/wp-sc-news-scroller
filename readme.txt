== Description ==

WP SC News Scroller plugin is a jquery based wordpress plugin which is used to show the wordpress news post with nice sliding effect from up to down. It can be integrated in the sidebar area on your Blog or Website. Anyone can free download the WP SC News Scroller plugin. WP SC News-Scroller plugin adds latest news scroller with a Title.

== Installation ==

1. Download the plugin.
2. Unzip the file.
3. Upload the "wp-sc-news-scroller" folder to the "/wp-content/plugins/" directory.
4. Activate the plugin through the "Plugins" menu in Wordpress.
5. Write the news or post by pressing the News-Scroller>New news menu in Wordpress.
6. And finally Drag the News-Scroller in your suitable widget area from Appearance>Widgets in Wordpress.
